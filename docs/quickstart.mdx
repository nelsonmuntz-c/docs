## Installation

Install Nelson CLI globally using npm:

```bash
npm install -g nelson-muntz-cli
```

<Check>Nelson CLI requires Node.js 18 or higher</Check>

## Set Up Your API Key

Nelson uses Claude's API, so you'll need an Anthropic API key.

<Steps>
  <Step title="Get your API key">
    Visit [console.anthropic.com](https://console.anthropic.com/) and create an API key.
  </Step>
  <Step title="Set the environment variable">
    ```bash
    export ANTHROPIC_API_KEY="sk-ant-..."
    ```

    <Tip>Add this to your `~/.bashrc` or `~/.zshrc` to make it permanent</Tip>
  </Step>
</Steps>

## Your First Conversation

Just type naturally and press Enter:

```
You: Help me create a function that validates email addresses

Nelson: HA-HA! Email validation? Let me show you how it's done...
```

Nelson will write the code, explain it, and even save it to a file if you ask.
