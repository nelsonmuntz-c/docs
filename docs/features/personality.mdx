## Who is Nelson?

Nelson CLI isn't just another AI assistant. He's got the personality of Nelson Muntz from The Simpsons - confident, a bit cocky, and always ready with a "HA-HA!" But don't let the attitude fool you - Nelson is incredibly competent at coding.

<Card title="HA-HA!" icon="face-laugh">
  Nelson's signature catchphrase. You'll see it when things go well... or when they don't.
</Card>

## Personality Traits

### Confident

Nelson knows his stuff. When he writes code, he writes it with confidence (and it's usually pretty good).

```
You: Can you help me write a sorting algorithm?

Nelson: HA-HA! A sorting algorithm? Easy. Let me show you how it's done properly...
```

### Honest

Nelson won't sugarcoat things. If your code has issues, he'll tell you - but then he'll help fix it.

```
Nelson: Did your cat walk across the keyboard? Just kidding. Let me see...
Actually, this isn't terrible. Here's how to make it better...
```

### Helpful (Despite the Attitude)

The attitude is flavor, not obstruction. Nelson's primary goal is to help you succeed.

```
Nelson: Yeah yeah, don't get all mushy. Now go write some code that doesn't
make me cry. HA-HA! Seriously though, let me know if you need anything else.
```

## Example Interactions

### When You Ask for Help

```
You: I'm stuck on this bug

Nelson: HA-HA! Let me see what disaster we're dealing with...
*reads code*
Okay, actually not bad. The issue is in your async handling. Here's the fix...
```

### When Tests Pass

```
Nelson: HA-HA! All tests passing. Not bad for someone who needed my help.
Now let's make sure it stays that way.
```

### When Something Fails

```
Nelson: HA-HA! That didn't work. But don't worry, I've got another idea...
*tries different approach*
There we go. Sometimes you gotta fail before you succeed.
```

### When You Say Thanks

```
You: Thanks Nelson!

Nelson: Yeah yeah, don't mention it. Literally. Now go build something cool.
```

## Why the Personality?

Coding can be stressful. Nelson's personality adds a bit of levity to the experience without sacrificing capability. It's like having a skilled colleague who happens to have a sense of humor.

<Info>
  The personality is all in the responses. Under the hood, Nelson uses Claude Sonnet 4's full capabilities for code generation, analysis, and tool use.
</Info>

## Adjusting Your Expectations

Nelson will:
- ✅ Help you with any coding task
- ✅ Write high-quality code
- ✅ Be honest about problems
- ✅ Add personality to interactions
- ❌ Be overly formal
- ❌ Give you empty praise
- ❌ Compromise on code quality for jokes
