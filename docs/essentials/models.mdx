## Claude Sonnet 4

Nelson CLI is powered by **Claude Sonnet 4**, Anthropic's most capable model for coding tasks.

<Card title="Claude Sonnet 4" icon="microchip">
  Fast, intelligent, and excellent at coding. The perfect balance of speed and capability.
</Card>

## Model Capabilities

### Code Generation

Nelson can write code in virtually any programming language:

- JavaScript / TypeScript
- Python
- Go
- Rust
- Java
- C / C++
- Ruby
- PHP
- And many more...

### Code Understanding

Nelson can read and understand existing codebases:

```
You: Read src/api/auth.js and explain the authentication flow
```

### Debugging

Nelson excels at finding and fixing bugs:

```
You: The tests are failing. Can you investigate and fix the issues?
```

### Architecture

Nelson can help design and implement software architecture:

```
You: Help me design a microservices architecture for an e-commerce platform
```

## Best Practices

<AccordionGroup>
  <Accordion title="Be Specific">
    The more specific your request, the better Nelson's response. Include details about language, framework, and requirements.
  </Accordion>
  <Accordion title="Provide Context">
    If you're working on an existing project, let Nelson read relevant files first to understand the codebase.
  </Accordion>
  <Accordion title="Iterate">
    Don't expect perfection on the first try. Work with Nelson iteratively to refine the output.
  </Accordion>
  <Accordion title="Use Nelson Loop">
    For complex tasks, use Nelson Loop to let the AI iterate automatically until completion.
  </Accordion>
</AccordionGroup>
