# `clawdia pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
clawdia pairing list whatsapp
clawdia pairing approve whatsapp <code> --notify
```
